## What is PaperKit?

PaperKit is a desktop application for managing academic PDF collections. It helps users rename research papers, compress PDFs, export metadata, generate citation files, detect duplicate PDFs, and extract embedded images.

PaperKit is designed to be safe by default. It does not modify original files unless the user explicitly enables direct renaming.

## Features

- Rename PDFs using extracted paper titles
- Preview filenames before applying changes
- Optional numbering with custom start number
- Detect exact duplicate PDFs
- Compress PDFs using Ghostscript
- Export metadata to Excel
- Export RIS and BibTeX citation files
- Extract selected embedded images from PDFs
- Drag-and-drop support
- Operation summaries and Excel reports
- Dark/light/system appearance modes
