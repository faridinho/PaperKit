# Release Notes

## PaperKit v1.2.0

Initial public release.

### Features

- Rename PDFs using extracted academic paper titles
- Preview filenames before renaming
- Optional numbering with custom start number
- Duplicate PDF detection
- PDF compression with Ghostscript
- Metadata export to Excel
- RIS and BibTeX citation export
- Embedded image extraction with preview and selection
- Drag-and-drop support
- Operation summaries and reports
- Dark, light, and system appearance modes

### Notes

PDF compression requires Ghostscript:

https://ghostscript.com/releases/gsdnld.html

Rename, metadata export, citation export, duplicate detection, and image extraction do not require Ghostscript.

Windows may show a SmartScreen warning because the app is not code-signed.
